Class: AddRemove

AddRemove()

new AddRemove()

Container for creating transactions. Lets the user withdraw or deposit items either by using manual forms or by submitting a picture for object recognition. Uses Hooks to fetch data, and submits POST-requests by using a submit function through the apiCall function
